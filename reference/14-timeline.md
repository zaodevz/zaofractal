# 14 - Master Timeline (2014 - 2026)

A single chronology of the fractal-governance movement, from Larimer's first blockchain to ZAO Fractal in May 2026.

| When | Event |
|------|-------|
| 2014 | Daniel Larimer launches **BitShares** - first decentralized exchange |
| 2016 | Larimer launches **Steem/Hive** - first blockchain social media with token rewards |
| 2018 | Larimer launches **EOS** (EOSIO) - $4B+ market cap at launch |
| **Feb 20, 2021** | Larimer publishes **"More Equal Animals: The Subtle Art of True Democracy"** (Amazon, Goodreads, Apple Books, Google Books, Kobo) |
| April 2021 | **Eden on EOS** launches - first practical fractal democracy implementation. 400+ peak members |
| Sept 6, 2021 | Larimer announces first Eden on EOS blockchain election. EOS Foundation grant (200K EOS + 200 EOS/member), 182 registrants |
| **Oct 9, 2021** (13:00 UTC) | **First on-chain fractal election** held on Eden on EOS |
| **Jan 28, 2022** | **Fractally protocol announced** via Medium post by Larimer |
| Feb 22, 2022 | **Fractally White Paper 1.0** published |
| 2022 | **Genesis Fractal** - 30-week experiment, ~130 participants, Dan SingJoy placed 3rd. Proves weekly Respect Game cadence works |
| **May 2022** | **Eden Fractal founded** (Epoch 1, on EOS). Dan SingJoy founder, ~10 weekly regulars |
| Feb 2023 | Tadas Vaitiekunas (`sim31`) releases the **original Fractalgram** (Telegram Web A client for live sessions) |
| ~2023 | **Fractally goes dormant**. Website stays up, no active development |
| **October 2023** | **Optimism Fractal launches** on OP Mainnet. Dan SingJoy + Optimystics team. ~65 Respect holders, bi-weekly cadence |
| Late 2023 | Eden Fractal restructured around formal mission/vision/strategy |
| **~Aug 2024** | **ZAO Fractal begins**. Zaal founder. Weekly Mondays 6pm EST cadence |
| Jul 30, 2024 | **ZAO OG Respect** ERC-20 deployed on Optimism: `0x34cE89baA7E4a4B00E17F7E4C0cb97105C216957` |
| Nov 2024 | **Optimism Fractal Council formally approves ORDAO** in Season 5. First production deployment of ORDAO on Ethereum mainnet |
| **June 5, 2025** | **Eden Fractal Epoch 2 launches on Base** at Event #121 (3-year anniversary). ORDAO deployed, EOS Respect migrated via snapshot + claim |
| Sept 11, 2025 | **ZAO ZOR Respect** ERC-1155 deployed on Optimism: `0x9885CCeEf7E8371Bf8d6f2413723D25917E7445c`. Fractal 74 becomes the dividing line: 1-73 = OG era (Airtable), 74+ = ZOR era (on-chain via OREC) |
| **January 2026** | **Optimism Fractal pauses indefinitely**. Council votes to consolidate into Eden Fractal. ZAO Fractal becomes the only active Optimism fractal and one of only two on the Superchain |
| Feb-Apr 2026 | `orclient` v1.4.4 published on npm (Apr 2, 2026). `orfrapps` repo active. `sim31/ordao` last push Apr 2, 2026 |
| Mar 6-12, 2026 | ZAO Fractal bot consolidated and rebranded as `fractalbotmarch2026` (v1.3 - v1.6). 52 slash commands. Timer overhaul |
| Mar 24, 2026 | Farcaster Foundation opens Discussion #19 - **FIP: Proof of Work Tokenization** by CassOnMars |
| Mar 27, 2026 | ZAO Fractal bot **v2.0**: migrates all storage from JSON to Supabase, wired into existing ZAO OS tables |
| **Mar 28, 2026** | ZAO Fractal bot **v2.1** (current): optional onchain auto-submit via hot wallet, Farcaster identity linking (`/link_farcaster`), Snapshot cog polling every 5 min |
| Apr 2, 2026 | sim31/ordao last commit. orclient v1.4.4 published on npm |
| Apr 24, 2026 | ZAOstock sprint-fractal adaptation designed (ZAO Doc 498). Three parallel 6-person fractals, bi-weekly cadence tied to festival milestones |
| May 17, 2026 | Async GitHub-native fractal brainstorm (ZAO Doc 664). Decision pending: pursue or stay synchronous |
| May 18, 2026 | Tanja x Zaal call. Documentation gap identified as #1 onboarding blocker. Fractal Circles research committed |
| **May 21, 2026** | ZAO Fractal current-state audit (ZAO Doc 703): **Week 100+, OREC at 242+ transactions, bot v2.1 live**, 2-wallet OREC bottleneck and dashboard rebuild flagged as open issues |

## What is *not* on this timeline

- Roy Fractal, Fractal Hispano, Alien Worlds Fractal, Aquadac, EOS Respect, ZEOS Fractal - all founded around 2022 in the wave following Fractally's launch. They run on their own schedules and have not been milestone-dated here. See [10-fractal-communities-directory.md](10-fractal-communities-directory.md).
- Internal Optimystics repo commits, ZAO research docs, individual session-by-session ZAO Fractal events. The timeline focuses on **public, external-facing milestones**.

The pre-2024 entries (Larimer's books, Fractally announcement, Eden on EOS election dates, Genesis Fractal stats) were independently verified May 21, 2026 against primary sources (Amazon, Medium, edenfractal.com, hive.blog).
